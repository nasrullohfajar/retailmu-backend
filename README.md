# Retailmu - Backend

Aplikasi manajemen retail berbasis web yang memungkinkan pengelolaan produk, kategori, supplier, transaksi, dan laporan keuangan secara efisien.

## Tech Stack
- Node.js + Express + TypeScript
- MongoDB + Mongoose
- JWT + Cookie
- Zod
- Bcrypt

## Fitur
- Autentikasi JWT via cookie HttpOnly
- Middleware protect & role-based access control
- Soft delete dengan audit trail (createdBy, updatedBy, deletedBy)
- Validasi request menggunakan Zod
- Global error handler dengan custom ApiError
